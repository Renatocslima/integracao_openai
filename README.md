## Detalhes do projeto

Esse código apresenta uma integração python com ChatGPT
  
                      
Para Rodar esse projeto será necessário:

```
-Criar uma conta na OpenAI
-Criar uma API_KEY
-Criar um arquivo .env com -> OPENAI_API_KEY: "sua chave"

instalar:

pip install python-dotenv
pip install openai

```
